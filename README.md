# WhatsApp Web Print

A Tampermonkey userscript that adds a floating print button for opened images on [WhatsApp Web](https://web.whatsapp.com).

## What it does

- Shows a print button when an image is open in WhatsApp Web.
- Opens a clean print view for the current image.

## Installation

Preferred method:

1. Open the script page on GreasyFork: [WhatsApp Web Print](https://greasyfork.org/cs/scripts/568147-whatsapp-web-print).
2. Click **Install this script**.
3. Confirm installation in Tampermonkey.

Manual method:

1. Install the **Tampermonkey** browser extension.
2. Create a new userscript.
3. Paste the content of `script.user.js`.
4. Save and reload [WhatsApp Web](https://web.whatsapp.com).

## Usage

1. Open an image in WhatsApp Web.
2. Click the green print button in the bottom-right corner.
3. Confirm printing in your browser print dialog.
